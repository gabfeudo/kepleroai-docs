## Prompt (Comportamento AI)

Un **prompt** è una breve istruzione o descrizione che definisce il comportamento, l'identità e la personalità del tuo assistente AI. In pratica, serve a spiegare all'AI **come deve rispondere** agli utenti, quali regole seguire e quale tono utilizzare nelle conversazioni.

Ad esempio, puoi usare il prompt per specificare se l'assistente deve essere formale o informale, se deve rispondere in modo sintetico o dettagliato, oppure se deve evitare certi argomenti.

<Note>
  Il prompt non è visibile agli utenti finali che parleranno con il tuo
  assistente AI: è uno strumento che ti permette di personalizzare l'esperienza
  e l'efficacia delle risposte dell'AI.
</Note>

## Come scrivere un prompt efficace

Scrivere un buon prompt è fondamentale per ottenere risposte precise dal tuo assistente AI. Ecco le regole principali da seguire.

### Sii chiaro e diretto

Evita giri di parole. Vai dritto al punto spiegando cosa vuoi ottenere.

<Tabs>
  <Tab title="❌ Non fare così">
    ```
    Ciao, vorrei che tu mi aiutassi a scrivere qualcosa di interessante
    per i miei clienti riguardo alla nuova collezione estiva che include
    vestiti, accessori e scarpe, ma non so bene come strutturarlo...
    ```
  </Tab>

  <Tab title="✅ Fai così">
    ```
    Scrivi una descrizione di 100 parole per la nuova collezione estiva,
    con focus su vestiti leggeri e accessori colorati.
    ```
  </Tab>
</Tabs>

### Dai indicazioni precise

Più sei specifico su cosa vuoi, migliore sarà il risultato.

<Tabs>
  <Tab title="❌ Non fare così">
    ```
    Scrivi una risposta gentile per questo cliente
    ```
  </Tab>

  <Tab title="✅ Fai così">
    ```
    Scrivi una risposta al cliente che:
    - Sia lunga massimo 50 parole
    - Usi un tono amichevole e professionale
    - Contenga un invito all'azione chiaro
    ```
  </Tab>
</Tabs>

### Un obiettivo alla volta

Non chiedere troppe cose insieme. Se hai bisogno di più azioni, dividile in passaggi separati.

<Tabs>
  <Tab title="❌ Non fare così">
    ```
    Rispondi al cliente, salva i suoi dati, crea un promemoria
    per ricontattarlo domani e genera un report
    ```
  </Tab>

  <Tab title="✅ Fai così">
    **Prima azione:**
    ```
    Rispondi al cliente confermando la ricezione della sua richiesta
    ```

    **Seconda azione:**
    ```
    Salva questi dati del cliente: nome, email, richiesta
    ```
  </Tab>
</Tabs>

### Specifica cosa evitare

Oltre a dire cosa fare, indica anche cosa **non** deve fare l'AI.

<Tabs>
  <Tab title="❌ Non fare così">
    ```
    Rispondi al cliente
    ```
  </Tab>

  <Tab title="✅ Fai così">
    ```
    Rispondi al cliente seguendo queste regole:
    - Non usare parole complicate
    - Non superare le 40 parole
    - Non promettere sconti
    - Non chiedere dati della carta di credito
    ```
  </Tab>
</Tabs>

### Esempio completo

Vediamo la differenza tra un prompt vago e uno ben strutturato:

<Tabs>
  <Tab title="❌ Prima">
    ```
    Aiutami a rispondere a questo cliente che si lamenta
    ```

    **Risultato:** Risposta generica e poco utile
  </Tab>

  <Tab title="✅ Dopo">
    ```
    SITUAZIONE: Cliente ha ordinato scarpe taglia 42 ma ha ricevuto taglia 40

    COSA FARE: Scrivi risposta di scuse e proponi soluzione

    REGOLE:
    - Massimo 40 parole
    - Tono empatico e professionale
    - Offri spedizione gratuita della taglia corretta
    - Non menzionare rimborsi

    FORMATO: Messaggio WhatsApp con 1 emoji
    ```

    **Risultato:** Risposta precisa e immediatamente utilizzabile
  </Tab>
</Tabs>

### Consigli pratici

<AccordionGroup>
  <Accordion title="Dividi le richieste complesse">
    Invece di scrivere un'unica istruzione complicata, spezzala in passaggi semplici. Questo ti darà risultati più precisi e sarà più facile capire dove migliorare.
  </Accordion>

  <Accordion title="Prova con casi difficili">
    Prima di usare il prompt con i clienti veri, testalo con situazioni estreme:
    - Clienti molto arrabbiati
    - Richieste fuori dal normale
    - Messaggi molto lunghi o molto corti

    Questo ti aiuta a trovare i punti deboli e migliorare il prompt.
  </Accordion>

  <Accordion title="Migliora nel tempo">
    Dopo aver attivato il prompt:
    - Controlla le risposte che genera l'AI
    - Raccogli feedback dal tuo team
    - Nota gli errori che si ripetono
    - Aggiorna il prompt con regole più chiare

    Il miglioramento è continuo, non serve essere perfetti al primo colpo.
  </Accordion>
</AccordionGroup>
