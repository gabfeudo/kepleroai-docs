## Come gestire le conversazioni

Keplero ti permette di gestire le conversazioni in vari modi, ad esempio: 
1. <a href="/dashboard/conversazioni/cartelle">Utilizzando le cartelle per raggruppare le conversazioni</a> 
2. <a href="/dashboard/conversazioni/etichette">Abbinando delle etichette alle varie chat</a>
3. <a href="/dashboard/conversazioni/filtri#filtrare-per-operatore">Assegnando le conversazioni a un operatore</a>
4. <a href="/dashboard/conversazioni/controllo-manuale">Prendendo il controllo manuale delle conversazioni</a>
5. <a href="/dashboard/conversazioni/salva-contatto">Salvando il contatto del cliente direttamente dalla conversazione</a>
6. <a href="/dashboard/conversazioni/elimina-chat">Eliminando le chat che non vuoi più vedere</a>
7. <a href="/dashboard/conversazioni/riassumi">Riassumendo le chat</a>
8. <a href="/dashboard/conversazioni/traduci">Traducendo i messaggi</a>
9. <a href="/dashboard/conversazioni/promemoria">Impostando un promemoria</a>
10. <a href="/dashboard/conversazioni/nuovo-messaggio">Iniziando una conversazione WhatsApp (outbound)</a> (*Solo se si ha integrato il canale WhatsApp*)
