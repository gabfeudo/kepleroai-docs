---
title: "Booking Expert"
description: "Collega Booking Expert a Keplero per fornire ai tuoi clienti link pre-compilati per la prenotazione"
---

<Steps>
    <Step title="Crea una nuova azione">
        1. Accedi alla sezione Addestramento nella dashboard di Keplero AI
        2. Clicca su **Integrazioni**
        3. Clicca su **+ Nuova integrazione**
        4. Seleziona **Hotel** dal menu laterale

        <Frame>
            <img src="/images/integrazioni/hotel/hotel-1.png" alt="Booking Expert" />
        </Frame>
    </Step>
    <Step title="Seleziona Booking Expert">
        Seleziona Booking Expert dall'elenco delle integrazioni disponibili.
    </Step>
    <Step title="Inserisci i dati necessari per collegarti a Booking Expert">
        <Frame>
            <img src="/images/integrazioni/hotel/booking-expert.png" alt="Booking Expert" />
        </Frame>
        Inserisci: 
        1. URL del motore
        2. Layout ID
        3. Adult Type ID
        4. ID Tipo Teen
        5. Child Type ID
        6. ID Tipo Bambino
    </Step>
</Steps>
