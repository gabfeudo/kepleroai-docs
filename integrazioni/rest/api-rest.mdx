# Creare un'integrazione API REST

Questa guida ti mostra come configurare una nuova integrazione REST API su Keplero AI, permettendo all'AI di invocare endpoint esterni e automatizzare azioni personalizzate.

<Steps>
    <Step title="Crea una nuova integrazione REST">
        Vai nella sezione <strong>Integrazioni</strong> e seleziona <strong>Nuova integrazione</strong> &rarr; <strong>REST</strong>.
    </Step>
    <Step title="Compila i dettagli dell'azione">
        Inserisci il <strong>nome</strong> dell'azione e una <strong>descrizione</strong> chiara e comprensibile. La descrizione aiuta l'AI a capire quando attivare questa integrazione.
    </Step>
    <Step title="Configura i parametri da estrarre">
        Definisci i parametri che l'AI dovr√† estrarre (es. <strong>note ordine</strong>, <strong>codice ordine</strong>). Fornisci descrizioni dettagliate e, se possibile, esempi pratici per ogni parametro.
    </Step>
    <Step title="Imposta la richiesta API">
        Scegli la <strong>tipologia di richiesta</strong> (GET, POST, ecc.), inserisci l'<strong>URL</strong> dell'endpoint, eventuali <strong>query parameter</strong>, <strong>header</strong> per l'autenticazione e il <strong>body</strong> della richiesta.

        <Tip>
            Nel body e negli altri campi puoi usare le variabili estratte racchiudendole tra parentesi graffe, ad esempio: <code>\{\{codice_ordine\}\}</code>.
        </Tip>
    </Step>
    <Step title="Salva e testa l'integrazione">
        Salva la configurazione e utilizza la sezione <strong>Esecuzioni</strong> per monitorare le chiamate API e il debug degli argomenti estratti e delle risposte.
    </Step>
</Steps>

## Debug e test

Durante i test, puoi vedere:
- L'icona lampo che indica l'invocazione dell'azione
- Gli argomenti estratti e inviati all'API
- La risposta ricevuta dall'endpoint
- Tutta la configurazione modificabile in tempo reale
